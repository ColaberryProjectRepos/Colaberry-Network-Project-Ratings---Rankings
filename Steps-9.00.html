To populate the dimension tables

First we need to create a stored procedure.



Run the attached zip file above to create The Procedures.

CreateSProcedures.sql

Second create an SSIS package  and rename Load_Dimension




To Create the connection to the source

Right click on Connections manager and select New OLE DB Connections .




To Create the connection to the destination

Right click on Connections manager and select New OLE DB Connections .



To Load DimUsers

Step 1 - Drag and drop Execute SQL Task to the control flow

Step 2 - Right click on the control flow and click on Edit



Step 3 - Make a connection to your SQL server By clicking the down arrow bottom

Step 4- Click on the SQL Statement line and click on the three dots

Below window will open


Insert below script and Click OK

EXECUTE [dbo].[proc_Load_DimUsers_SCD_Type2]


Step 5 - Click OK

To Load the rest of the Dimensions

you will Repeat the above step 1 to Step 5 for each dimensions

But you need to change the Execute statement on Step 4

For DimPreUser

 

EXECUTE [dbo].[proc_Load_DimPreUser_SCD_Type2]

 

For DimTags

 

EXECUTE [dbo].[proc_Load_DimTag_SCD2]

 

For DimTrackingSource

 

EXECUTE [dbo].[proc_Load_DimTrackingSource_SCD2]

 

For DimProjects

 

EXECUTE [dbo].[proc_Load_DimProjects_SCD2]
