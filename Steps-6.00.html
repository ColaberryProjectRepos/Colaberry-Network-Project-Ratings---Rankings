Create Databases required for the Project 6

These Steps will be performed in SQL Server Management Studio (SSMS)

  1. Right Click on Databases, then click on New Database.
  2. Name the Database. (ColaberryNetwork_staging)
  3. Click Ok to create the Database, then refresh Database to locate New Database Created.



Repeat this step for all those 3 steps we are looking to create.
